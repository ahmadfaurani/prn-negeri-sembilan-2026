Ryanair mengumumkan keuntungan suku tahunannya merosot lebih satu pertiga susulan kenaikan mendadak harga bahan api penerbangan akibat perang Iran serta tindakan syarikat penerbangan itu menurunkan tambang bagi merangsang permintaan, lapor PA Media/dpa.
Syarikat penerbangan tambang rendah dari Ireland itu pada Isnin melaporkan keuntungan selepas cukai susut 34 peratus kepada €538 juta (AS$615.9 juta) bagi tempoh tiga bulan berakhir Jun.
Menurut syarikat itu, prestasi kewangannya terjejas akibat lonjakan harga bahan api penerbangan bagi 20 peratus keperluan bahan api yang tidak dilindung nilai, selain penurunan purata tambang sebanyak enam peratus.
Keadaan itu mengimbangi pertumbuhan enam peratus jumlah penumpang kepada 61.3 juta orang serta peningkatan satu peratus hasil keseluruhan kepada €4.38 bilion.
Ketua Pegawai Eksekutif Michael O'Leary berkata syarikat itu mengambil langkah menurunkan tambang kerana "konflik di Asia Barat menyebabkan pengguna lebih berhati-hati untuk melakukan perjalanan, kebimbangan terhadap kekurangan bekalan bahan api penerbangan di Kesatuan Eropah (EU), ketidaktentuan ekonomi serta tempahan yang dibuat semakin hampir dengan tarikh penerbangan."
Harga bahan api penerbangan meningkat dua kali ganda kepada AS$150 setong pada suku berkenaan apabila perang Iran menyebabkan harga minyak dan gas melonjak, selain penutupan Selat Hormuz.
Keadaan itu turut menyebabkan kos operasi Ryanair meningkat 11 peratus kepada €3.42 bilion pada suku berkenaan.
Perjanjian damai sementara antara Amerika Syarikat (AS) dan Iran bulan lepas memberikan sedikit kelegaan kepada harga minyak dan tenaga, namun harga kembali melonjak selepas rundingan gagal mencapai persetujuan dan pertempuran tercetus semula.
O'Leary berkata masih terlalu awal untuk memberikan unjuran prestasi syarikat bagi keseluruhan tahun kerana keputusannya "amat sensitif terhadap perkembangan luaran yang tidak menggalakkan, termasuk peningkatan konflik di Asia Barat dan Ukraine, harga bahan api penerbangan yang tidak dilindung nilai, kejutan makroekonomi serta mogok berterusan dan kelemahan pengurusan kawalan trafik udara di Eropah."
Syarikat penerbangan yang berpangkalan di Dublin itu berkata tambang terus mencatatkan penurunan sederhana pada suku kedua walaupun tempahan meningkat sedikit kebelakangan ini, dengan penumpang masih cenderung membuat tempahan hampir dengan tarikh penerbangan.
"Walaupun jumlah penumpang pada musim panas 2026 kekal kukuh, tempoh tempahan masih lebih hampir dengan tarikh penerbangan berbanding tahun lepas, sekali gus mengehadkan lagi keterlihatan terhadap prestasi masa depan," kata O'Leary.
"Seperti kebiasaannya pada peringkat awal tahun, kami masih belum mempunyai gambaran yang jelas bagi separuh kedua tahun ini. Oleh itu, masih terlalu awal untuk memberikan sebarang unjuran bermakna mengenai keuntungan selepas cukai bagi tahun kewangan 2026-2027."
